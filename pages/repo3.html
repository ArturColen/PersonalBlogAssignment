Repositório: CraftAPI

Descrição

Projeto desenvolvido na disciplina Framework do curso técnico de desenvolvimento web e mobile com o objetivo de consolidar os conceitos aprendidos sobre React Hooks, bem como reforçar o conteúdo de consumo de APIs.
O projeto consiste em listar todos os mobs, equipamentos e minérios no Minecraft e exibir detalhes sobre eles usando a API desenvolvida pela dupla.
